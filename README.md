# 高中小論文：智慧農場與氣候變遷

📄 本研究探討科技農業如何因應氣候變遷問題，並透過實地訪查與田野觀察，分析智慧農場在當代農業系統中的角色與挑戰。

---

## 🧠 研究主題核心
- 人文地理與現代科技的結合
- 智慧農場系統的實際應用
- 氣候變遷對農業科技的影響

---

## 🔍 研究過程與方法
- 小組主題發想與文獻回顧
- 實地訪查智慧農場，自主設計訪談提綱
- 訪問農場經營者，觀察科技設備運作
- 撰寫與彙整觀察記錄，製作小論文成果
